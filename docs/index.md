# Wakt

**Take back control of your screen time.**

Wakt is a free, open-source digital wellness app for Android that helps you block distracting apps and stay focused.

## Features

- **App Blocking** - Block distracting apps with customizable challenges
- **Scheduled Locks** - Set automatic phone/app locks on a schedule
- **Focus Sessions** - Quick-start focus timers (25min, 1hr, 2hr)
- **Lock Commitment** - Lock your schedules to prevent easy changes
- **Privacy First** - All data stays on your device

## Privacy First

- No data collection
- No analytics or tracking
- No advertisements
- Fully open source
